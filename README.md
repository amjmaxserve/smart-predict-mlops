# SmartPredict Kubernetes Platform

## Overview

This branch contains the Kubernetes-native deployment architecture for the SmartPredict MLOps platform.

Planned components:

- FastAPI Inference API
- MLflow Tracking
- MinIO Object Storage
- Prometheus Monitoring
- Grafana Dashboards
- Kubernetes Deployments
- Services
- Ingress
- ConfigMaps
- Secrets
- Persistent Volumes

---
